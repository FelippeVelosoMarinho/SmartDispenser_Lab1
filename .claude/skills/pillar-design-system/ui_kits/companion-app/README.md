# Companion app — UI kit

Patient-facing mobile app. Pairs with the Pillar dispenser. Shows today's schedule, lets the user log doses (including ones taken away from the device), and surfaces gentle reminders.

## Core screens
- **Today** — primary home. The full day at a glance, closest dose highlighted.
- **Log dose** — confirmation sheet after taking a pill.
- **Medication detail** — what it is, why, refill status.
- **Caregivers** — people in the loop. You choose what they see.

Open `index.html` to click through a prototype of the full flow. Components live as `.jsx` files alongside.
